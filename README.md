# myTorch

从零实现一个最小可训练的深度学习框架，核心目标是理解自动求导与训练流程。

## 项目目标

- 手写动态计算图与反向传播（autograd）
- 逐步实现 `Tensor` 的基础运算与梯度回传
- 在不依赖主流深度学习框架的前提下，搭建可训练的最小系统

## 当前范围

- 第一阶段：标量级 `Tensor` 与 `backward()`
- 后续阶段：张量化运算、模块化网络、优化器与训练循环

## 目录结构

- `core/`：核心数据结构与算子定义
- `tests/`：单元测试与梯度正确性验证
- `run_tests.py`：测试统一入口

## 当前进度

- 已完成项目骨架搭建
- 正在实现 `Tensor` 与基础算子逻辑

## 开发原则

- 先保证正确性，再扩展功能
- 每增加一个算子，配套补充测试
- 保持接口清晰、代码可读
